# Linux 終端快捷鍵



| **^C     終止前台運行的程序**                 |
| :-------------------------------------------- |
| **^D    退出等同于exit**                      |
| **^L     清屏**                               |
| **^A    光標移到命令行的最前端**              |
| ^E     光標移到命令行的最后端                 |
| ^U     刪除光標前所有字符                     |
| ^K      刪除光標后所有字符                    |
| ^R      搜索歷史命令                          |
| ESC .  引用上個命令的最後一個參數 等價與  !$  |
| ALT .   引用上個命令的最後一個參數 等價與  !$ |
